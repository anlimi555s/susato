# 寿里美化新版本适配:
接手寿里美化更新。

请，请，请不要使用库里的美化版本。请去release下载发布版本，库是用来堆资料和子mod以及备份的，其中的susato-model不一定是你需要的版本。

>更新请去[release](https://github.com/anlimi555s/susato/releases)

关于寿里美化未来命名标准，以此为例：susato-model-v5.10-0.5。

其中v5.10表示适配游戏版本v0.5.10。0.5表示美化适配版更新版本号。

受限于个人产能原因，未来更新采用的是社区提交制，如果您想将自己绘画和改版的部分提交到新版本更新，可以前往汉化组discord上找到susato-model的频道，或者加入QQ群540579363来发送消息。

同时下载适配版请去[release](https://github.com/anlimi555s/susato/releases)

仓库用于存放mod和作为未来更新计划的地方，不再直接存放mod，可以打开查看bug以及一些适配版问题与画饼。

>模组安装简单轻便，直接加载入模组加载器即可。注意：请勿加载多个美化。
