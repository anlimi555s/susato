# 理发店渲染回退补丁

## 概述

将 DoL 新版 (Jun 2026) 理发店的 `mannequinHairdresser` 渲染回退到旧版 (Feb 2026) 行为。

**补丁类型**：Twee (ReplacePatcher)  
**目标段落**：`Hairdressers Widgets`  
**日期**：2026-07-05 :P

---

## 新旧版差异对比

| 渲染特性 | 新版 (Jun 2026) | 旧版 (Feb 2026) |
|----------|:--:|:--:|
| 隐藏 base 层 | `hidelayer "base"` | 无 |
| 手臂 | `arm_left/right = "none"` | 默认显示 |
| 伤疤 | `scars = false` | 无控制 |
| 转化部件预览 | 50 行 chimera 逻辑 | 无 |
| show_tf 开关 | `show_tf = $transformationShow` | 无 |
| 穿着 | `worn.over_upper[3]` | `worn.upper[14]` + `worn.lower[15]` 白色 |

### 代码级差异

**新版** (`<<selectmodel "main">>` 内部)：
```
<<hidelayer "base">>
<<set _modeloptions.arm_left to "none">>
<<set _modeloptions.arm_right to "none">>
<<set _modeloptions.tanningEnabled to false>>
<<set _modeloptions.scars to false>>
...
/* Automate this later ... */
[50行 TF chimera 配置]
<<set _modeloptions.show_tf = $transformationShow>>
<<set _modeloptions.worn.over_upper.index to 3>>
```

**旧版** (相同位置)：
```
<<set _modeloptions.tanningEnabled to false>>
...
<<set _modeloptions.worn.upper.index to 14>>
<<set _modeloptions.worn.upper.colour to "white">>
<<set _modeloptions.worn.lower.index to 15>>
<<set _modeloptions.worn.lower.colour to "white">>
```

## 补丁实现

### 技术细节

- **补丁串长度**：from 8,970 字符 → to 3,329 字符
- **匹配方式**：精确字符串替换，从 `<<selectmodel "main">>` 到 `<<rendermodel "canvas-mannequin">>` 之前的完整模型配置块
- **缩进**：使用 `\t` (Tab) 和 `\n` (LF) 精确匹配 Twee 源码
- **段落定位**：`Hairdressers Widgets` — 所有理发店相关 widget 定义的集中段落

### 调试记录

| 尝试 | 问题 | 解决 |
|------|------|------|
| 初版 | `passageName: "Hairdressers Seat"` | widget 定义在 `Hairdressers Widgets`，调用才在 `Hairdressers Seat` |
| 验证 | `from` 字符串在 `Hairdressers Widgets` 段落中精确匹配确认 | 通过 Node 脚本逐字符验证 |

### 提取脚本

用于从 HTML 提取精确匹配字符串的命令：

```js
// 解码 HTML 实体
const decoded = html.replace(/&lt;/g, '<').replace(/&gt;/g, '>').replace(/&quot;/g, '"');

// 提取 mannequinHairdresser widget
const match = decoded.match(/<<widget "mannequinHairdresser">>([\s\S]*?)<<\/widget>>/);

// 提取 <<selectmodel "main">> 到 <<rendermodel "canvas-mannequin">> 之间的内部块
const inner = match[1].match(/(<<selectmodel "main">>[\s\S]*?)<<rendermodel "canvas-mannequin">>/);

// JSON.stringify 得到精确的 JSON 转义字符串
const escaped = JSON.stringify(inner[1]);
```
